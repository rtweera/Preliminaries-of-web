# Preliminaries of Web

A comprehensive collection of HTML learning activities designed to build foundational web development skills. This repository contains progressive HTML exercises that teach core concepts of web page structure, semantics, styling, and layout.

## 📚 Project Overview

This repository is an educational resource for beginners learning web development. It contains four progressive HTML activities that introduce and reinforce key web development concepts, from basic HTML structure to more complex layouts and styling.

**Target Audience**: Beginners with no prior web development experience

**Level**: Beginner (HTML fundamentals)

## 🚀 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/rtweera/Preliminaries-of-web.git
   cd Preliminaries-of-web
   ```

2. **Open an activity in your browser**:
   - Simply open any HTML file directly in your web browser
   - Or use a local server (e.g., `python -m http.server` or `npx http-server`)

3. **Follow the learning path**:
   - Start with Activity 1 and progress through Activity 4
   - Review the code and understand each concept
   - See LEARNING_PATH.md for guided progression

## 📖 Activities Overview

| Activity | Topic | Skills Covered |
|----------|-------|-----------------|
| **Activity 1** | Basic HTML Structure | DOCTYPE, html, head, body, headers, paragraphs, links, images |
| **Activity 2** | Semantic HTML & Lists | Description lists (dl, dt, dd), text formatting, structure |
| **Activity 3** | Layout & Positioning | Image positioning, alignment, improved semantic structure |
| **Activity 4** | CSS Styling | Internal CSS, styling, tables, footer layouts, responsive design |

For detailed information about each activity, see [ACTIVITIES.md](ACTIVITIES.md).

## 🎯 Learning Outcomes

By completing these activities, you will understand:

- ✅ HTML document structure and proper DOCTYPE declaration
- ✅ Semantic HTML elements and their proper usage
- ✅ Working with images and managing image assets
- ✅ Creating structured content with headers, paragraphs, and lists
- ✅ HTML form elements and their attributes
- ✅ Internal CSS styling and basic layout techniques
- ✅ Web accessibility best practices
- ✅ Code organization and best practices
